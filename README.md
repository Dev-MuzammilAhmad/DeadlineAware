# ⏰ DeadlineAware (Deadline Tracker)

A modern, mobile-responsive deadline tracking application designed to help users organize and monitor tasks. It features automated failure state resolution for overdue tasks, custom status confirmations, clean KPI visualizations, and secure database persistence.

### 🛠️ Technology Stack
* **Frontend**: React 19, Vite 8, Tailwind CSS v4, React Router v7
* **Database & Auth**: Firebase SDK (Authentication & Firestore)
* **Email System**: EmailJS Browser SDK
* **Charts & Icons**: Recharts, Lucide React
* **Testing**: Vitest + React Testing Library

### 🚀 Getting Started
Run `npm install` to load dependencies, copy `.env.example` to `.env` with your API credentials, and start the app with `npm run dev`.
